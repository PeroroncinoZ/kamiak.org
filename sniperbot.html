Commands #

pls dox
Finds someone's school picture, school, and student id
pls dox name
pls dox student_id
pls dox teacher_id
Details
Only supports the Mukilteo School District
Student/teacher IDs may not be available for all schools
An alternative command is pls doxx
snipe
Brings back the most recently deleted message
snipe
pls snipe
Details
Must be used in the same channel
Can be used multiple times to further bring back deleted messages
Sniper Bot can snipe itself if a moderator deletes its snipe response message
kanye
Provides a random Kanye West quote
kanye
west
Details
Uses the kanye.rest api to fetch random tweets from Kanye
Activates when a message containing "kanye" or "west" is sent
